防封号A模式：AXB

  • 产品介绍

    A（员工手机号）——> X（中间号）——>B（客户）：直接拨打电话或者点击拨号，A 直接拨号每次都会拨打同一个 X 号码，然后由系统拨打B号码，客户手机显示为员工真实号码。

  • 防封号原理

    员工手机同一天只对单个号码进行呼叫，大大降低对外多个号码的呼叫频率，封号概率大大下降。

    附：因业务投诉导致封号除外。

  • 产品特点

    1. 1、使用灵活：点击号码呼叫，直接输入号码呼叫；
    2. 2、外呼频次限制不大、不封、不停号码；
    3. 3、通话质量有保证，不受网络宽带限制；
    4. 4、业务系统使用便捷；
    5. 5、后台管理系统；

  • 其他功能

    ①支持用户自定义信息字段 ②支持数据导入 ③录音查听和下载 ④工作量统计 ⑤支持API对接

防封号B模式：双呼

  • 产品介绍

    A（员工手机号）<——X（中间号）——>B（客户）：直接拨打电话或者点击拨号后，会收到X号的来电，当A接通后，X会再向B发起呼叫，B接听后A和B可 以正常通话。(A和B都作为被叫，被封概率非常低)

  • 防封号原理

    双呼模式使得员工主叫和用户号码都成为被叫号码，无外呼频率所以不会被封号。

    附：因投诉导致封号除外。

  • 产品特点

    1. 1. 使用灵活：点击号码呼叫，直接输入号码呼叫;
    2. 2. 外呼频次无限制、不封、不停号码;
    3. 3. 通话质量有保证，不受网络宽带限制;
    4. 4. 业务系统使用便捷
    5. 5. 后台管理系统
